## 📋 Pré-requisitos

Antes de executar os projetos, certifique-se de ter instalado:

- **Node.js** (versão recomendada: 18 ou superior)
- **npm** ou **yarn**
- Aplicativo **Expo Go** no seu celular ou um emulador configurado (Android Studio para Android, Xcode para iOS)

Para verificar as versões instaladas:

```bash
node -v
npm -v
```

## ⚙️ Como executar os projetos

Para rodar qualquer um dos aplicativos acima:

1. Entre na pasta do projeto desejado:
   ```bash
   cd nome-do-projeto
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor do Expo:
   ```bash
   npx expo start
   ```

4. Escaneie o QR Code com o seu celular ou pressione:
   - `a` para Android
   - `i` para iOS

## 🛠️ Criação de um Novo Projeto

Para criar um novo projeto Expo do zero:

```bash
npx create-expo-app meu-novo-app
cd meu-novo-app
npm run reset-project  # Para resetar o projeto inicial
npx expo install @expo/vector-icons  # Para ícones
```
